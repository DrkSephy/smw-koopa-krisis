Super Mario World: Koopa Krisis


Move left/right: LEFT/RIGHT ARROW ; Climb up/down ladders: UP/DOWN ARROW

Jump (Hold to jump higher): X ; Use Powerup: C ; Use pipes/doors, Select levels: SPACEBAR

What is Super Mario World Koopa Krisis?

Super Mario World: Koopa Krisis is a game created to learn more about ImpactJS and JavaScript, while trying to build a starter RPG game.
Our next installment will feature a 2D side scrolling action RPG, built using the features within this game.
